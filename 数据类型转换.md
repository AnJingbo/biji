

**当一个整形字面量没有超出byte、short、char的取值范围，这个字面值可以直接赋值给byte、short、char的变量。**

```java
byte b = 97;
short s = 97;
char c = 97;
```

但是**当整型变量赋值给byte、short、char变量的时候，还是需要强制类型转换，否则可能会损失精度，从而报错。**
```java
int i = 97;
byte b = i;// 报错
short s = i;// 报错
char c = i;// 报错
```

另：Java中的整形字面量有三种表示形式：
 * 十进制，是一种缺省的默认方式
 * 八进制，在编写八进制整数型字面值的时候需要以0开始
 * 十六进制，在编写十六进制整数型字面值的时候需要以0x开始

```java
int a = 123;
int b = 0123;// 83
int c = 0x123;// 291
```
